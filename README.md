¿Qué es Angular?
Es un framework creado por Google para armar aplicaciones web. Imaginalo como una fábrica profesional: ya tienes las máquinas y los procesos listos para que tu app sea robusta, escalable y no se rompa a la primera.


¿Qué herramientas necesitas?
Solo este combo básico:

Node.js: El motor para que todo funcione (descárgalo de su web oficial).

Angular CLI: La herramienta que instalas desde la terminal para crear y manejar tus proyectos.

Visual Studio Code: Tu editor para escribir el código.

TypeScript por una razón sencilla: orden y seguridad.
